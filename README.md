## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Config / Secrets environment variables

Copy `.env.example` to `.env` and add your private information

*Note: never commit this file, it should be ignored by git*

```
DISCORD_TOKEN="<your-token>"
DISCORD_PREFIX="!"
```

## Installation

#### SSH

```bash
$ git clone git@github.com:EddieHubCommunity/EddieBot.git
```

#### GitHub CLI

```bash
$ gh repo clone EddieHubCommunity/EddieBot
```

#### HTTPS

```bash
$ git clone https://github.com/EddieHubCommunity/EddieBot.git
```


```bash
$ cd EddieBot
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
